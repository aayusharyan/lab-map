# Getting Help

## Documentation

- **Docker setup**: [docker/README.md](docker/README.md) — volumes, environment variables, deployment options
- **Configuration format**: [CONTRIBUTING.md](CONTRIBUTING.md) — data file structure, validation, and schema details

## Bug Reports

Found something broken? Open an issue using the **Bug Report** template:

[Report a bug](https://github.com/aayusharyan/lab-map/issues/new?template=bug-report.yml)

Before opening, check whether the issue already exists in [open issues](https://github.com/aayusharyan/lab-map/issues).

## Feature Requests

Have an idea for an improvement? Use the **Feature Request** template:

[Request a feature](https://github.com/aayusharyan/lab-map/issues/new?template=feature-request.yml)

## Icon Requests

Need an icon for a device type that isn't supported yet?

[Request an icon](https://github.com/aayusharyan/lab-map/issues/new?template=icon-request.yml)

Or contribute one directly — see the [Creating Node Icons](CONTRIBUTING.md#creating-node-icons) section of the contributing guide.

## Security Issues

Do **not** open a public issue for security vulnerabilities. See [SECURITY.md](SECURITY.md) for private disclosure options.
